# StreamHost

Shares a monitor or a single window to other people's browsers. Windows only.
They watch in the built-in viewer or by opening a link in Chrome or Firefox.

## Build from source

Needs the .NET 10 SDK. From the repo root:

    dotnet run --project src/StreamHost

## Run without building

Download the latest build from the Releases page, unzip, and run StreamHost.exe.
It is self-contained, so you do not need .NET installed.
